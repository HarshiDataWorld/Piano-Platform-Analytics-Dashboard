# Piano Platform Analytics Dashboard
## Preview
![Dashboard](dashboard_preview.png)
## Overview
End-to-end analytics project for a global online piano learning platform
operating across 10+ countries with 5,000+ users.

## Tools Used
Power BI | Python | Excel | SQL

## Key Insights
- Identified time zone misalignment as primary dropout driver
- Proposed scheduling optimization → projected 30% retention improvement
- Revenue grew from $19K → $38.4K over 12 months

## Dashboard Features
- KPI cards: Revenue, Users, Retention, Dropout
- Monthly revenue & enrollment trend
- Country-wise user breakdown
- Dropout analysis by time zone gap
- Interactive Region & Month slicers
